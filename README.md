# todo_list

Список команд:\
create_list - создать файл со списком задач\
delete_list - удалить файл со списком задач\
add <название задачи> - добавить задачу в список\
delete <номер задачи> - удалить задачу из списка\
set_priority <номер задачи> - изменить приоритет задачи\
list - вывести весь список задач\
done <номер задачи> - пометить задачу выполненой\
undone <номер задачи> - пометить задачу не выполненой\
priority - показать все невыполненные задачи от наивысшего приоритета до наименьшего\
clear - очистить список задач\
exit - выход из приложения

Формат файла:\
{\
id (long long int) - номер задачи\
name (string) - название задачи\
completed (bool) - статус выполнения задачи (0 - не выполнен, 1 - выполнен)\
description (string) - описание задачи\
priority (int) - приоритет задачи (0 - наивысший приоритет, 10 - наименьший приоритет)\
}
